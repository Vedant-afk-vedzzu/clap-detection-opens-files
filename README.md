# clap-detection-opens-files
Clap.py is a lightweight Python script that listens for audio input from your microphone and detects double claps. When two claps occur within a set time window, the script executes a customizable system command â€” for example, opening a website, launching an application, or running another script.
## Features
- Detects claps in real time using your microphone
- Executes any system command on double clap
- Configurable sensitivity and timing
- Lightweight and beginner-friendly
